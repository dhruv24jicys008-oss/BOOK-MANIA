# BOOK-MANIA
The Books Mania is a responsive and interactive book showcase website built using HTML, CSS, and JavaScript. 
It highlights featured books, allows users to search for books, and provides direct purchase links.

# Features

1. Modern UI with gradients and smooth animations

2. Fully responsive design (mobile & desktop friendly)

3. Search functionality to quickly find books

4. Scroll-to-top button for better navigation

5. AOS (Animate On Scroll) animations

# How Search Works

1. Users enter the book name in the search box

2. JavaScript maps the input to a book ID

3. If found, the page smoothly scrolls to the book

4. If not found, an alert message is shown
